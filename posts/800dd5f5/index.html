Terminal 常用指令

發表於 2022-05-27 分類於 Cheat Sheet 文章字數： 512 所需閱讀時間 ≈ 1 分鐘

在使用指令介面時，偶爾會忘記特定指令。所以建立一篇 cheat sheet 方便之後需要時查詢。

快速鍵

Windows 快速鍵 MacOS 快速鍵 用途
tab tab 輸入部分，可自動補齊後面內容
↑ ↓ ↑ ↓ 跳出之前用過的指令，可直接 enter 執行

常用指令

Windows 指令 MacOS 指令 用途
cd .. cd .. 進入上層資料夾
cd path cd path 進入指定資料夾
clear clear 清空終端機內容
ls ls 列出當前目錄裡的所有內容
ls -a ls -a 列出當前目錄裡的所有內容(含隱藏檔案)
mkdir folder name(s) mkdir folder name(s) 新建一個或多個資料夾
open . start . 打開當前目錄的圖形化介面
pwd pwd 列出當前位置路徑
rm file name(s) rm file name(s) 刪除一個或多個檔案
rm -rf folder name(s) rm -rf folder name(s) 刪除一個或多個資料夾(及其內容)
(r = recursive, f = force)
touch file name(s) touch file name(s) 新建一個或多個檔案

備註：在命令列進行「刪除」不會將檔案/資料夾移到資源回收筒，而是直接刪除。
