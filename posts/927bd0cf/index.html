HTML 介紹

我們俗稱的「網頁」，指的就是 HTML 文件經過瀏覽器解析後呈現的結果。製作網頁很簡單，只要建立一個 .html 文件，將想要呈現的東西寫進去再用瀏覽器打開就可以了。

HTML 的全稱是 Hypertext Markup Language。它是一種 「標記語言 (markup language)」，而不是 「程式語言 (programming language)」。標記語言是一種專門把內容結構化、資訊化的語言，為我們寫的東西賦予某種特性。

HTML 註解

如果想要在 .html 文件裡寫下備註說明，但又不想干擾程式碼時，就可以使用註解 (comment)。HTML 的註解是用 <!-- --> 來包圍一段內容，被註解包圍的內容會被瀏覽器忽略。

HTML 標籤 (tag) 是什麼?

HTML 標籤是一個對原始內容進行說明，讓瀏覽器知道怎麼呈現的工具。

如果把瀏覽器比喻成一個廚師，現在我們提供廚師 3 顆雞蛋，然後在分別在雞蛋上貼上寫著「蒸蛋」、「荷包蛋」和「水煮蛋」的便利貼，最後廚師就會將相同的 3 顆生雞蛋煮成 3 道不同的料理。這 3 顆生雞蛋指的就是 .html 文件裡的原始內容，而便利貼指的就是 HTML 標籤，它負責告訴瀏覽器該對原始內容進行怎樣的處理。

HTML 標籤基本語法

標籤的使用

  • 在 < > 內寫入標籤名稱來定義這個標籤。
  • 除了部分例外，標籤都是成對的，包含一個開始標籤和一個結束標籤，而要套用這個標籤的原始內容則會被包在開始標籤和結束標籤中間。結束標籤在標籤名稱前須加上一個 /。
  • HTML 標籤可以形成巢狀的關係，一層包一層。
  • 一組被標籤包圍的網頁內容，我們會叫它 網頁元素 (element) 。下面例子: <p> 是 <strong> 的父元素；<strong> 是 <p> 的子元素。

標籤的屬性

在開始標籤的標籤名稱後加入一個空白鍵，可接著為標籤寫入屬性，並在 ="" 內寫入屬性值。例如: href 標示連結的目標網址、style 可為個別標籤加上 CSS 樣式。

除了每種標籤本身可套用的屬性外，我們常會為標籤加上 class 或 id 屬性。

  • 當我們寫了很多標籤後，如果要對某一些標籤做樣式設定或選取，一個一個單獨設定是非常沒有效率的，所以可以用 class 為標籤分類，然後直接對這個分類進行操作即可(例如:改變字體顏色…等)。 class 的值不限一個，用一格空白鍵分開即可。
  • id 則是給特定標籤設定一個獨一無二的名字，方便之後對指定標籤做操作。例如: 一個網頁裡可能有很多 <h1> 標籤，當我要設定一個頁內跳轉連結，點一下後直接跳到第二個 <h1> 標籤位置。我可以為第二個 <h1> 標籤加入 id ，變成 <h1 id="jump-here"> ，然後將連結指向 jump-here 即可。
  • 另外還有一個比較特殊的自定義屬性 data-*，這是標準 HTML 所沒有的，而是開發者因為需求自行定義的屬性，通常在 “data-“ 後加上自定義名稱。例如: JS30 Day2 使用了 data-key="65" ，當我們按下鍵盤”A” (keycode:65) 時，網頁要給出特定回應。

HTML 標籤

HTML 使用語意化標籤，看標籤名稱就能大致知道它的用途。

網頁結構相關標籤

  • <head> - 紀錄網頁相關資訊，如:網頁名稱、使用的語言、文字編碼…等。
  • <body> - 瀏覽器畫面看得見的內容都放在這裡面。

下列標籤都是放在 <body> 裡的。

傳統作法是使用 <div> 並搭配 id 或 class 來定義網頁結構(例如: <div class=”nav”> )，後面慢慢發現類似的結構不斷出現，所以到 HTML5 在 2014 年發佈時，就提供了更多的語義化元素。

  • <header> - 網頁標頭。
  • <nav> - 導覽列 (navigation)，通常放入功能選單連結。
  • <main> - 放入網站主要內容。
  • <section> - 區塊，定義內容分類架構。
  • <article> - 文章，常用於 blog 型內容網站。
  • <aside> - 側邊欄，主內容相關的次級資訊。
  • <footer> - 頁尾，通常放入作者資訊或版權。

常用標籤

  • heading - <h1> ~ <h6> 總共六級的標題標籤，愈有意義的資訊，就應該使用愈高的標題層級。
  • <br> - 斷行、換行 (break)。
  • <div> - 分割區塊 (division)。除傳統用法外(上方網頁結構相關標籤說明)，多用來作為 CSS 樣式設定的容器使用。
  • <p> - 段落文字 (paragraph)。
  • <ol> - 有序清單 (ordered list)，會呈現有編號的清單。
  • <ul> - 無序清單 (unordered list)，用小黑點取代編號的清單。
  • <li> - 清單項目 (list)，為 <ul> 和 <li> 的子元素，每個清單項目需用 <li></li> 包裹。

其他標籤

  • <a> - 是指 anchor
  • <audio> - 幫網頁加上聲音 ( 基本介紹 、 JS30範例:JavaScript Drum Kit )
  • <img> - 是指 image ， 幫網頁加上圖片
